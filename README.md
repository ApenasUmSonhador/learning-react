# Learning React

A project to learn and practice React fundamentals with hands-on exercises.

## Overview

This repository documents my journey through learning React, starting with the basics and progressing to more advanced concepts. The focus is on understanding core React principles through practical implementation.

## What's Included

### Completed
- **React Basics**: Components, JSX, and rendering
- **Interactive Components**: State management with `useState`
- **Props**: Passing data between components
- **Event Handling**: Managing user interactions
- **Component Composition**: Building reusable components

### Projects
- **Tic-Tac-Toe Game**: A complete interactive game implementation based on [React's official tutorial](https://react.dev/learn/tutorial-tic-tac-toe) featuring:
  - Game board with 9 squares
  - Turn-based gameplay (X and O players)
  - Winner detection
  - Game history and time travel functionality
  - Move navigation

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```

The application will open in your browser at `http://localhost:3000`
